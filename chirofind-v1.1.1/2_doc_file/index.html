Installation

Follow the steps below to setup your site template:

  1. Unzip the downloaded chirofind package and open the /chirofind folder to find all the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  2. Below is the folder structure and needs to be uploaded to your website or localhost root directory:
    • chirofind/assets - Contains all of the assets referenced
      • assets/css - Stylesheet files
      • assets/images - Images files
      • assets/js - Javacript files
      • assets/inc - PHP files
      • assets/vendors - All external libs.
    • chirofind/index.html - Home Page 1
    • chirofind/index2.html - Home Page 2
    • chirofind/index3.html - Home Page 3
    • chirofind/index4.html - Home Page 4
    • chirofind/about.html - About Us Page
    • chirofind/services1.html - Service 1
    • chirofind/services2.html - Service 2
    • chirofind/chiropractic-care.html - Service details
      • chirofind/trained-therapists.html - Trained Therapists
      • chirofind/massages-therapy.html - Massages Therapy
      • chirofind/chiropractic-care.html - Chiropractic Care
      • chirofind/laser-therapy.html - Laser Therapy
      • chirofind/physical-rehab.html - Physical Rehab
    • chirofind/portfolio.html - Portfolio
    • chirofind/portfolio-details.html - Portfolio Details
    • chirofind/team1.html - Team 1
    • chirofind/team2.html - Team 2
    • chirofind/team-details.html - Team Details
    • chirofind/pricing.html - Pricing Table
    • chirofind/Appointment.html - Appointment Form
    • chirofind/faq.html - Faq Page
    • chirofind/404.html - 404 Eerror Page
    • chirofind/shop.html - Shop Page
    • chirofind/shop-details.html - Shop Details Page
    • chirofind/cart.html - Cart Page
    • chirofind/wishlist.html - Wishlist Page
    • chirofind/checkout.html - Checkout Page
    • chirofind/blog.html - Blog Page
    • chirofind/blog-list.html - Blog List Page
    • chirofind/blog-details.html - Blog Details Page
    • chirofind/contact.html - Contact Page
  3. You should upload all or specific HTML files as per your need.
  4. You are good to go for adding your content now!

HTML Structure

Chirofind follows a simple and easy to customize coding structure. Here is the sample for your reference: 


            <!DOCTYPE html>
            <html lang="en">
            <head>
              <!-- Stylesheet
              ============================== -->
              <!-- Bootstrap -->
              <link rel="stylesheet" type="text/css" href="assets/vendors/bootstrap/css/bootstrap.min.css" />
              <!-- Font Awesome Icon -->
              <link rel="stylesheet" type="text/css" href="assets/vendor/font-awesome/css/all.min.css" />
              <!-- Magnific Popup -->
              <link rel="stylesheet" type="text/css" href="assets/vendor/magnific-popup/magnific-popup.min.css" />
              <!-- Main Stylesheet -->
              <link rel="stylesheet" type="text/css" href="assets/css/chirofind.css" />
              <link rel="stylesheet" type="text/css" href="assets/css/chirofind-responsive.css" />

            </head>

            <body>
              <!-- Main Wrapper   
              =============================== -->
              <div class="page-wrapper">  
                <!-- Header -->
                <header class="main-header">
                ......
                </header>
                <!-- Header End --> 
                
                <!-- Footer -->
                <footer class="site-footer">
                  ......
                </footer>
                <!-- Footer end --> 
              </div>
              <!-- Main Wrapper end --> 
              <!-- JavaScript -->
            </body>
            </html>

Theme Customization

How to Change Logo

Logo Container can be found in the Header Container - <header>


            <!-- Logo -->
            <div class="main-menu__logo">
              <a href="index.html">
                <img src="assets/images/resources/logo-1.svg" alt="FigTheme">
              </a>
            </div>
            <!-- Logo End -->

How to Change Copyright

Copyright Container can be found in the Footer Container - <footer>


            <!-- Copyright Text -->
            <div class="site-footer__bottom">
              <div class="container">
                <div class="site-footer__bottom-inner">
                  <p class="site-footer__bottom-text">©Copyright Chirofind <span id="currentYear"></span>.</p>
                  <ul class="list-unstyled site-footer__bottom-menu">
                    <li><a href="about.html">Trams & Condition</a></li>
                    <li><a href="about.html">Privacy Policy</a></li>
                  </ul>
                  <a href="#" data-target="html" class="scroll-to-target scroll-to-top"><i class="fas fa-arrow-up"></i></a>
                </div>
              </div>
            </div>
            <!-- Copyright Text End -->
